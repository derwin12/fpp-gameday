# fpp-gameday

## GameDay Scoring Plugin

This plugin is designed for [Falcon Pi Player (FPP)](https://github.com/FalconChristmas/fpp) and supports NFL, NCAA Football, NHL, MLB, and AFL games. It monitors live scores via the ESPN API and triggers FPP light sequences when a tracked team scores or wins.
